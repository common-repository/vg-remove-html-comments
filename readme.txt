== Description ==
VG Remove html comments is very simple and low weight plugin which helps WordPress/dashboard user to remove html comment from website. This MAY have an adverse impact on caching

== Installation ==
*	Download the plugin .zip file.
*	Login in to admin Click Plugins -> Add New -> Upload.
*	Find VG Remove html comment in plugin list  and activate it.
